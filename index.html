Welcome to MyIchimoku.eu

MyIchimoku.eu is a web application built to help trading stock by the use of Ichimoku Kinko Hyo charts. It is for my personal use really.

Its purpose is to store historical stock data, collect daily updates and analyse the data based on the Ichimoku trading method. The application provides suggestions for symbols that look attractive based on the technical analysis, provides buy and sell signals, contains manageble lists of favorites, actual portfolio and historical portfolio as well as an interactive trading plan.

Making this work vailable to the public also means that I would be greatly interested in your participation improving the code as this really is my first ever PHP/MySQL script. So BEWARE if and when you try to install the scripts, it WILL require a some efforts, but once it is running, it is running :-) Have a look at the website and the screenshots available there to convince yourself.

As it analyses the Ichimoku method in small logical steps and creates 'signal points' for each step, I introduced a 'MyIchimoku Signal' which summarises the strength on a daily basis. The application uses standard open source software such as PHP, MySQL, JQuery (and a number of it's plug-in's) and probably a few others. Thanks!! :-) .

In addition to this Yahoo! YQL is used for stock data feeds, though some historical data was provided by ABC Bourse. The JQuery charting plugin comes from Highcharts (great tool really, you should have a look at their website). Icons are provided by Doubje-J Design.

Last but not least: the Ichimoku trading basics where provided by Manesh Patel, by use of his book which would never have happened if I wouldn't have had an impuls purchase of this book: Secrets de trading d'un moine bouddhiste from Daniel Alleman. MERCI Daniel! After 20 years of absence on the stock trading market you got me back on track :-)
